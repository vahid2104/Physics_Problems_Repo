Uniform and Uniformly Accelerated Motion

This page explains the motion equation step by step, shows how to find velocity and acceleration, and visualizes the motion for the values x₀ = 0, v₀ = 5 m/s, and a = -2 m/s². It is designed to be simple, visual, and easy to present.

Main equation: x(t) = x₀ + v₀t + 1/2 at²

What we need to determine

  • The velocity function v(t)
  • The acceleration function a(t)
  • The stopping time
  • The maximum velocity
  • The maximum displacement
  • The graphs of x(t), v(t), and a(t)
Position → derivative → Velocity
Velocity → derivative → Acceleration
Stopping time when v(t)=0

Given values

x₀ = 0
v₀ = 5 m/s
a = -2 m/s²

Because the acceleration is negative, the object slows down over time. It will stop for a moment, then if we continue mathematically, it starts moving back.

1

Find the velocity

Velocity is the derivative of position with respect to time.

v(t) = dx/dt = v₀ + at

For the given values:

v(t) = 5 - 2t
2

Find the acceleration

Acceleration is the derivative of velocity with respect to time.

a(t) = dv/dt = a

For this problem, acceleration is constant:

a(t) = -2 m/s²
3

Find the stopping time

The object stops when the velocity becomes zero.

5 - 2t = 0
t = 2.5 s

So the object stops after 2.5 seconds.

Position function

x(t) = x₀ + v₀t + 1/2 at²

This tells us where the object is at time t. The x₀ part is the starting position, the v₀t part shows motion due to initial velocity, and the 1/2 at² part shows the effect of acceleration.

Velocity function

v(t) = v₀ + at

Velocity changes linearly with time. If a > 0, the object speeds up. If a < 0, the object slows down. In our main example, acceleration is negative, so velocity decreases.

Acceleration function

a(t) = a

Acceleration is constant in uniformly accelerated motion. That is why the acceleration graph is a horizontal line. A constant negative value means the object is continuously losing speed.

Main example explained clearly

x(t) = 5t - t²
v(t) = 5 - 2t
a(t) = -2

At the beginning, the object moves forward because the initial velocity is positive. But the constant negative acceleration reduces the velocity every second. After 2.5 s, the velocity becomes zero, so the object stops for a moment. At that same moment, the position reaches its highest value, which is 6.25 m.

This is why the position graph is a parabola with a highest point, the velocity graph is a decreasing straight line, and the acceleration graph is a constant horizontal line.

Tip for presentation: move the time slider slowly and explain how position, velocity, and acceleration are connected through derivatives.
