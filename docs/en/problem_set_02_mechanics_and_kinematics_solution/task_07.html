Problem 7 – 2D Motion with a Given Acceleration

We are given the constant acceleration

a = (2, -3)

and the initial conditions

v(0) = (1, 0) r(0) = (0, 0)

Step 1 – Velocity function

Since acceleration is the derivative of velocity, we integrate:

v'(t) = a = (2, -3) v(t) = (2t + C1, -3t + C2)

Using the initial condition v(0) = (1,0), we get:

C1 = 1 C2 = 0

So the velocity is:

v(t) = (2t + 1, -3t)

Step 2 – Position function

Position is the integral of velocity:

r'(t) = v(t) = (2t + 1, -3t) r(t) = (t² + t + C3, -(3/2)t² + C4)

Using the initial condition r(0) = (0,0), we get:

C3 = 0 C4 = 0

So the position is:

r(t) = (t² + t, -(3/2)t²)

Step 3 – Interpretation

  • The particle starts at the origin.
  • It initially moves to the right because v(0) = (1,0).
  • The acceleration always points right and downward.
  • The path is a curved trajectory (a parabola).
