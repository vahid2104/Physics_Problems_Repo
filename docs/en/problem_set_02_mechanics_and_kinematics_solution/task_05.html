Elliptical Motion – Purely Kinematic

This interactive page explains the motion defined by x(t) = a cos(ωt) and y(t) = b sin(ωt). You can see the trajectory, the moving particle, and the graphs of speed and acceleration magnitude.

1. Given motion

The particle moves according to the parametric equations:

x(t) = a cos(ωt)
y(t) = b sin(ωt)

Here:

  • a is the horizontal semi-axis of the ellipse
  • b is the vertical semi-axis of the ellipse
  • ω controls how fast the particle moves around the ellipse

If we eliminate the parameter t, we get the trajectory:

x²/a² + y²/b² = 1

So the path is an ellipse.

2. Velocity

Velocity is the derivative of position with respect to time.

v(t) = (dx/dt, dy/dt)

Differentiate each component:

vₓ(t) = -aω sin(ωt)
vᵧ(t) = bω cos(ωt)

So the velocity vector is:

v(t) = (-aω sin(ωt), bω cos(ωt))

The magnitude of velocity is:

|v(t)| = ω √(a² sin²(ωt) + b² cos²(ωt))

This means the speed is not constant in general. It becomes constant only when a = b, which means the ellipse becomes a circle.

3. Acceleration

Acceleration is the derivative of velocity with respect to time.

a(t) = (dvₓ/dt, dvᵧ/dt)

Differentiate again:

aₓ(t) = -aω² cos(ωt)
aᵧ(t) = -bω² sin(ωt)

So the acceleration vector is:

a(t) = (-aω² cos(ωt), -bω² sin(ωt))

The magnitude of acceleration is:

|a(t)| = ω² √(a² cos²(ωt) + b² sin²(ωt))

This value also changes with time, unless the motion is circular.

4. Where is the velocity maximum?

We use

|v(t)| = ω √(a² sin²(ωt) + b² cos²(ωt))

The largest speed appears where the larger coefficient contributes most.

  • If a > b, then the maximum speed occurs when sin²(ωt) = 1, so the particle is at (0, ±b).
  • If b > a, then the maximum speed occurs when cos²(ωt) = 1, so the particle is at (±a, 0).

Therefore, in general:

|v|max = ω max(a, b)

5. What the graphs show

The page contains three animated views:

  • Ellipse canvas: shows the trajectory and the moving particle
  • Speed graph: shows how |v(t)| changes with time
  • Acceleration graph: shows how |a(t)| changes with time

You can change a, b, and ω with the sliders and immediately see how the motion changes.
