Problem 2 – Projectile Motion

A body moves in the Earth's gravitational field without air resistance. It is launched with initial velocity v₀ at an angle α relative to the horizontal.

1. Theory

First, we split the initial velocity into horizontal and vertical components:

v₀x = v₀ cos(α) v₀y = v₀ sin(α)

Horizontal motion

There is no horizontal acceleration:

aₓ = 0 x(t) = v₀ cos(α) · t

Vertical motion

The vertical acceleration is caused by gravity:

aᵧ = -g y(t) = v₀ sin(α) · t - ½gt²

Time of flight

T = (2v₀ sin(α)) / g

Maximum height

Hmax = (v₀² sin²(α)) / (2g)

Range

R = (v₀² sin(2α)) / g

Maximum range

The range is maximum when:

α = 45°

3. Step-by-step explanation

  • The projectile starts with speed v₀ at angle α.
  • The horizontal velocity stays constant because there is no air resistance.
  • The vertical velocity changes because gravity pulls the body downward.
  • The body reaches its maximum height when the vertical velocity becomes zero.
  • The projectile lands when the vertical position returns to zero.
  • The angle for maximum range is 45°.
