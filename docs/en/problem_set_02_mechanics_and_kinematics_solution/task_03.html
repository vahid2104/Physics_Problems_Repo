Problem 3 – Elimination of Time and Interpretation of Acceleration

This interactive page explains the parametric motion x(t)=2t² and y(t)=3t³ in a simple step-by-step way.

Given:

x(t) = 2t²,     y(t) = 3t³

We will: eliminate the parameter t, identify the trajectory, compute the velocity and acceleration, and decide whether the acceleration is constant.

1. Eliminate the parameter \( t \)

Start from the equation for \( x \):

x = 2t²

Solve for \( t² \):

t² = x / 2

Now use the equation for \( y \):

y = 3t³

Square both sides:

y² = 9t⁶

Since \( t⁶ = (t²)³ \), substitute \( t² = x/2 \):

y² = 9(x/2)³ = 9x³ / 8

So the Cartesian equation is:

y² = (9/8)x³

2. Interpret the trajectory

The equation

y² = (9/8)x³

describes a semicubical parabola.

  • \( x = 2t² \ge 0 \), so the curve exists only for \( x \ge 0 \).
  • \( y = 3t³ \), so \( y \) may be positive, negative, or zero.
  • The curve passes through the origin \( (0,0) \).
  • It is symmetric with respect to the \( x \)-axis.

Therefore, the trajectory is a curve opening to the right.

3. Velocity vector \( \vec{v}(t) \)

The velocity is the derivative of the position:

\vec{v}(t) = (dx/dt, dy/dt)

Differentiate each coordinate:

dx/dt = 4t

dy/dt = 9t²

So the velocity vector is:

\vec{v}(t) = (4t, 9t²)

4. Magnitude of the velocity

The speed is the magnitude of the velocity vector:

|\vec{v}(t)| = √[(4t)² + (9t²)²]
|\vec{v}(t)| = √[16t² + 81t⁴]

Factoring \( t² \):

|\vec{v}(t)| = |t|√(16 + 81t²)

5. Acceleration vector \( \vec{a}(t) \)

The acceleration is the derivative of the velocity:

\vec{a}(t) = (d²x/dt², d²y/dt²)

Differentiate again:

d²x/dt² = 4

d²y/dt² = 18t

So the acceleration vector is:

\vec{a}(t) = (4, 18t)

6. Magnitude of the acceleration

|\vec{a}(t)| = √[4² + (18t)²]
|\vec{a}(t)| = √[16 + 324t²]

7. Is the acceleration constant?

No, it is not constant.

This is because:

\vec{a}(t) = (4, 18t)

The second component depends on t, so the acceleration changes with time.

Its magnitude also changes:

|\vec{a}(t)| = √[16 + 324t²]

Therefore, the acceleration is not constant.

Final Answers

y² = (9/8)x³
\vec{v}(t) = (4t, 9t²)
|\vec{v}(t)| = √(16t² + 81t⁴) = |t|√(16 + 81t²)
\vec{a}(t) = (4, 18t)
|\vec{a}(t)| = √(16 + 324t²)

The trajectory is a semicubical parabola opening to the right.

The acceleration is not constant.
