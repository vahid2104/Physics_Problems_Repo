Harmonic Motion – Motion Parameters

Step-by-step presentation of the task: period, frequency, maximum velocity, maximum acceleration, and numerical calculations for A = 0.2 m and f = 2 Hz.

x(t) = A cos(ωt + φ)

1. Given function

We start from the harmonic motion formula:
x(t) = A cos(ωt + φ)
Here:
A = amplitude
ω = angular frequency
φ = phase constant
x(t) = position at time t

2. Period and frequency

One full oscillation happens when the angle of cosine changes by:
2π
So:
ωT = 2π
Solve for the period:
T = 2π / ω
Frequency is the number of oscillations per second:
f = 1 / T
Substitute T:
f = ω / 2π
Final formulas:
T = 2π / ω
f = ω / 2π

3. Maximum velocity

Velocity is the derivative of position:
v(t) = dx/dt
Differentiate:
x(t) = A cos(ωt + φ)
v(t) = -Aω sin(ωt + φ)
Since the sine function is always between -1 and 1, the largest possible magnitude is 1.
|sin(ωt + φ)| ≤ 1
Therefore, the maximum velocity is:
vmax = Aω
Maximum velocity:
vmax = Aω

4. Maximum acceleration

Acceleration is the derivative of velocity:
a(t) = dv/dt
Differentiate:
v(t) = -Aω sin(ωt + φ)
a(t) = -Aω² cos(ωt + φ)
Since cosine is also between -1 and 1, its largest magnitude is 1.
|cos(ωt + φ)| ≤ 1
Therefore, the maximum acceleration is:
amax = Aω²
Maximum acceleration:
amax = Aω²

5. Numerical calculations

Given:
A = 0.2 m f = 2 Hz

Step 1: Calculate ω

ω = 2πf
ω = 2π · 2 = 4π rad/s ≈ 12.57 rad/s

Step 2: Calculate vmax

vmax = Aω
vmax = 0.2 · 4π = 0.8π m/s ≈ 2.51 m/s

Step 3: Calculate amax

amax = Aω²
amax = 0.2 · (4π)² = 3.2π² m/s² ≈ 31.58 m/s²
Angular frequency
ω ≈ 12.57 rad/s
Maximum velocity
vmax ≈ 2.51 m/s
Maximum acceleration
amax ≈ 31.58 m/s²

6. Interpretation + animation

The period tells us how long one full oscillation takes.
T = 1 / f = 1 / 2 = 0.5 s
So the object completes one full oscillation in 0.5 seconds.
The maximum velocity happens when the object passes through the middle position.
The maximum acceleration happens at the ends of the motion, where the object turns back.
Final answers:
T = 0.5 s
f = 2 Hz
ω = 4π rad/s ≈ 12.57 rad/s
vmax = 0.8π m/s ≈ 2.51 m/s
amax = 3.2π² m/s² ≈ 31.58 m/s²
