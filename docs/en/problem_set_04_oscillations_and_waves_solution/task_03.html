Q Task

The wave equation is given by

y(x,t) = A sin(kx - ωt)

We need to determine:

  • the wavelength λ,
  • the phase velocity v,
  • the value of v for k = 4π and ω = 20π,
  • whether the point x = λ oscillates in phase with the point x = 0.

1 Determine the wavelength

In the wave equation

y(x,t) = A sin(kx - ωt)

the quantity k is called the wave number.

The relation between wave number and wavelength is

k = 2π / λ

Now solve for λ.

kλ = 2π λ = 2π / k
Therefore, the wavelength is: λ = 2π / k

i Interpretation

The wavelength tells us the distance in space after which the wave repeats its shape.

So if one crest is at some position, the next crest appears one wavelength later.

That is why points separated by one full wavelength have the same oscillation pattern.

2 Determine the phase velocity

The phase velocity is the speed of a point with constant phase.

So we take

kx - ωt = constant

Differentiate both sides with respect to time:

d/dt (kx - ωt) = 0
k(dx/dt) - ω = 0

Since

dx/dt = v

we get

kv - ω = 0 kv = ω v = ω / k
Therefore, the phase velocity is: v = ω / k

i Interpretation

This velocity tells us how fast the wave pattern moves.

For example, a crest or a trough moves with the phase velocity.

Because the equation has the form kx − ωt, the wave travels in the positive x-direction.

3 Numerical calculation

We are given

k = 4π ω = 20π

Use the formula

v = ω / k

Substitute the values:

v = (20π) / (4π)

The factor π cancels:

v = 20 / 4 = 5
Therefore, v = 5 m/s

i Also find λ for these values

λ = 2π / k = 2π / 4π = 1/2
So the wavelength is: λ = 0.5 m

4 Does x = λ oscillate in phase with x = 0?

We compare the displacement at both points.

At x = 0:

y(0,t) = A sin(0 - ωt) = A sin(-ωt)

At x = λ:

y(λ,t) = A sin(kλ - ωt)

But

k = 2π / λ

so

kλ = 2π

Therefore

y(λ,t) = A sin(2π - ωt)

Since the sine function repeats every 2π,

sin(2π - ωt) = sin(-ωt)

Thus

y(λ,t) = y(0,t)
Yes. The point x = λ oscillates in phase with the point x = 0.

i Physical meaning

Two points are in phase when they always have the same displacement at the same time.

Points separated by one full wavelength behave identically.

That is exactly why x = 0 and x = λ are in phase.
