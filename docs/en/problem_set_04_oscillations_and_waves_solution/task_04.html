Problem 4 – Wave Equation

We want to check whether the function y(x,t)=A cos(kx − ωt) satisfies the wave equation and then find the relation between v, k, and ω.

Given function
y(x,t)=A cos(kx−ωt)
Wave equation
∂²y/∂t² = v² ∂²y/∂x²

To verify this, we must:

  • find the second derivative with respect to time
  • find the second derivative with respect to position
  • substitute both into the wave equation

1. Derivative with respect to time

Start with:
y(x,t)=A cos(kx−ωt)
First derivative:
∂y/∂t = Aω sin(kx−ωt)
Second derivative:
∂²y/∂t² = −Aω² cos(kx−ωt)

So the time derivative result is −Aω² cos(kx−ωt).

2. Derivative with respect to position

Start again with:
y(x,t)=A cos(kx−ωt)
First derivative:
∂y/∂x = −Ak sin(kx−ωt)
Second derivative:
∂²y/∂x² = −Ak² cos(kx−ωt)

So the position derivative result is −Ak² cos(kx−ωt).

3. Substitute into the wave equation

∂²y/∂t² = v² ∂²y/∂x²
Left side:
∂²y/∂t² = −Aω² cos(kx−ωt)
Right side:
v² ∂²y/∂x² = v²[−Ak² cos(kx−ωt)]
So we get:
−Aω² cos(kx−ωt) = −Av²k² cos(kx−ωt)

The common factor −A cos(kx−ωt) appears on both sides, so we cancel it.

4. Algebraic relation

After cancellation:
ω² = v²k²
Take square root:
ω = vk
Solve for wave speed:
v = ω/k

Final relation

v = ω/k

Equivalent form: ω = vk

5. Final answer

The function y(x,t)=A cos(kx−ωt) satisfies the wave equation

∂²y/∂t² = v² ∂²y/∂x²

because its second derivatives are:

Time derivative

∂²y/∂t² = −Aω² cos(kx−ωt)

Position derivative

∂²y/∂x² = −Ak² cos(kx−ωt)

Substituting them gives:

ω² = v²k²

Therefore, the required algebraic relation is:

v = ω/k
