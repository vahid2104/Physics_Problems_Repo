1. Equation of motion

Start from Newton’s second law

In one dimension, Newton’s second law is:

F = ma
a = d²x / dt²
m(d²x / dt²) = F(x)
m(d²x / dt²) = −kx
d²x / dt² + (k/m)x = 0
Meaning

The force is proportional to position, but with a minus sign. That means the force always points back toward the origin. This is called a restoring force.

So the body does not keep moving away forever. Instead, it oscillates around x = 0.

2. Solve the equation of motion

Define angular frequency
ω = √(k/m)
Then the equation becomes:
d²x / dt² + ω²x = 0

This is the standard equation of simple harmonic motion.

General solution
x(t) = A cos(ωt) + B sin(ωt)
So:
x(t) = A cos(√(k/m) t) + B sin(√(k/m) t)

The constants A and B depend on the initial position and initial velocity.

3. Work done from 0 to x₀

Formula for variable force

In one dimension, work done by a variable force is:

W = ∫0x₀ F(x) dx
W = ∫0x₀ (−kx) dx
W = −k ∫0x₀ x dx
W = −k [x²/2]0x₀
W = −(1/2)kx₀²
Interpret the sign

The work is negative. That happens because when the body moves to the right, the force points to the left. So the force opposes the displacement.

In simple words: the spring resists being stretched.

4. Interpret the result as potential energy

Conservative force idea

For a conservative force, the change in potential energy is:

ΔU = −W
ΔU = (1/2)kx₀²

If we choose U(0) = 0, then the potential energy function becomes:

U(x) = (1/2)kx²
Meaning

At the origin, the potential energy is minimum. As the object moves farther from the origin, the potential energy increases.

So the spring stores more energy when it is stretched or compressed more.

5. Verify that F = −dU/dx

Differentiate U(x)
U(x) = (1/2)kx²
dU/dx = d/dx [(1/2)kx²]
dU/dx = kx
−dU/dx = −kx
Conclusion

But the given force is:

F(x) = −kx

Therefore:

F(x) = −dU/dx

The relationship is verified correctly.

Final answers

m(d²x/dt²) = −kx
d²x/dt² + (k/m)x = 0
x(t) = A cos(√(k/m) t) + B sin(√(k/m) t)
W = −(1/2)kx₀²
U(x) = (1/2)kx²
F(x) = −dU/dx
  • The motion is simple harmonic motion.
  • The force is a restoring force.
  • The work from 0 to x₀ is negative.
  • The corresponding potential energy is quadratic in x.
