Newton’s Second Law - Constant Force

Animated explanation and full solution for the motion of a body under a constant force.

Problem Data

m = 2 kg
F = (6, 2) N
v(0) = (1, -1) m/s
r(0) = (0, 0) m

We want to determine:

  • Acceleration vector a(t)
  • Velocity vector v(t)
  • Position vector r(t)
  • Trajectory of the motion
  • Work done by the force at t = 3 s
  • Verification of the work-energy theorem

1. Acceleration

From Newton’s second law:

F = m a

So:

a = F / m

Substitute the values:

a = (6, 2) / 2
a = (3, 1) m/s²

Therefore:

a(t) = (3, 1) m/s²

2. Velocity

The velocity is:

v(t) = v(0) + a t

Substitute the known values:

v(t) = (1, -1) + (3, 1)t
v(t) = (1 + 3t, -1 + t)

Therefore:

v(t) = (1 + 3t, -1 + t) m/s

3. Position

The position is:

r(t) = r(0) + v(0)t + (1/2) a t²

Substitute the values:

r(t) = (0,0) + (1,-1)t + (1/2)(3,1)t²
r(t) = (t + 3/2 t², -t + 1/2 t²)

Therefore:

r(t) = (t + 1.5t², -t + 0.5t²) m

4. Trajectory of the Motion

The position components are:

x(t) = t + 1.5t²
y(t) = -t + 0.5t²

This means the motion in the plane is not a straight line. The body first moves downward because the initial vertical velocity is negative, then it goes upward because the vertical acceleration is positive.

The trajectory is a parabola.

5. Work Done by the Force at t = 3 s

Work done by a constant force is:

W = F · Δr

First find the position at t = 3:

r(3) = (3 + 1.5·9, -3 + 0.5·9)
r(3) = (3 + 13.5, -3 + 4.5)
r(3) = (16.5, 1.5)

Since the initial position is (0,0), the displacement is:

Δr = (16.5, 1.5)

Now calculate the dot product:

W = (6,2) · (16.5,1.5)
W = 6·16.5 + 2·1.5
W = 99 + 3
W = 102 J

6. Check the Work-Energy Theorem

The theorem says:

W = ΔEₖ

Initial kinetic energy

v(0) = (1,-1)
|v(0)|² = 1² + (-1)² = 2
Eₖ₀ = (1/2)m|v(0)|²
Eₖ₀ = (1/2)·2·2 = 2 J

Kinetic energy at t = 3

v(3) = (1+3·3, -1+3)
v(3) = (10, 2)
|v(3)|² = 10² + 2² = 104
Eₖ(3) = (1/2)·2·104 = 104 J

Now calculate the change in kinetic energy:

ΔEₖ = Eₖ(3) - Eₖ₀
ΔEₖ = 104 - 2
ΔEₖ = 102 J

Since:

W = 102 J
ΔEₖ = 102 J

we get:

W = ΔEₖ

So the work-energy theorem is satisfied.

Final Answers

a(t) = (3, 1) m/s²
v(t) = (1 + 3t, -1 + t) m/s
r(t) = (t + 1.5t², -t + 0.5t²) m
Trajectory: parabola
Work at t = 3 s: 102 J
Work-energy theorem: verified
