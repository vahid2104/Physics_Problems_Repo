1. The problem

We study a body thrown vertically upward while gravity and air resistance act on it. The drag force is proportional to the velocity, so the equation is linear.

m dv/dt = -mg - kv

The term -mg is gravity, and the term -kv is drag. Both act downward while the object moves upward.

dv/dt + (k/m)v = -g

This is a first-order linear differential equation.

2. Solve for the velocity

We solve the equation step by step: first the homogeneous part, then a particular solution, and finally apply the initial condition.

Homogeneous equation

dv/dt + (k/m)v = 0

Its solution is

vh(t) = Ce-(k/m)t

Particular solution

Because the right side is constant, we try a constant solution:

vp = A

Substitute into the equation:

(k/m)A = -g   ⇒   A = -mg/k

General solution and initial condition

v(t) = Ce-(k/m)t - mg/k

Use the condition v(0) = v₀:

v₀ = C - mg/k   ⇒   C = v₀ + mg/k
Final velocity formula:

v(t) = (v₀ + mg/k)e-(k/m)t - mg/k

3. Solve for the position

Since velocity is the derivative of position, we integrate the velocity function.

dx/dt = v(t)
x(t) = ∫[(v₀ + mg/k)e-(k/m)t - mg/k] dt

After integration we get

x(t) = -(m/k)(v₀ + mg/k)e-(k/m)t - (mg/k)t + C₁

Now use the condition x(0) = 0.

C₁ = (m/k)(v₀ + mg/k)
Final position formula:

x(t) = (m/k)(v₀ + mg/k)(1 - e-(k/m)t) - (mg/k)t

4. Maximum height

The maximum height happens when the upward velocity becomes zero.

Time to reach the top

v(t) = 0
(v₀ + mg/k)e-(k/m)t - mg/k = 0
e-(k/m)t = mg / (kv₀ + mg)
tmax = (m/k) ln(1 + kv₀/(mg))

Maximum height

Substitute t = tmax into the position formula.

xmax = mv₀/k - (m²g/k²) ln(1 + kv₀/(mg))
This is smaller than the no-drag case.

5. Compare with motion without drag

If there is no drag, the only force is gravity. Then the formulas become much simpler.

With drag

v(t) = (v₀ + mg/k)e-(k/m)t - mg/k
xmax = mv₀/k - (m²g/k²)ln(1 + kv₀/(mg))

The speed decreases faster, so the body reaches a lower height and stops rising earlier.

Without drag

v(t) = v₀ - gt
x(t) = v₀t - (1/2)gt²
xmax = v₀² / (2g)

The object rises higher because there is no extra resistive force.

We use Euler’s method with example values to compare the numerical result with the exact analytical solution.

7. Final answers

These are the main formulas and conclusions of the task.

Velocity: v(t) = (v₀ + mg/k)e-(k/m)t - mg/k
Position: x(t) = (m/k)(v₀ + mg/k)(1 - e-(k/m)t) - (mg/k)t
Time of maximum height: tmax = (m/k)ln(1 + kv₀/(mg))
Maximum height: xmax = mv₀/k - (m²g/k²)ln(1 + kv₀/(mg))
Without drag: xmax = v₀²/(2g)
Conclusion: drag lowers the maximum height and the numerical solution agrees well with the analytical one.
