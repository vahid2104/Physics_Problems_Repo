Problem 5 – Momentum and One-Dimensional Head-On Collision

This page explains, step by step, how to solve an elastic collision of two bodies moving on one straight line. It includes the formulas, special cases, physical meaning, and a small animated demo.

1. Conservation laws

Conservation of momentum

In one-dimensional motion, the total momentum before and after the collision stays the same.

m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂

Here:

  • u₁, u₂ = velocities before collision
  • v₁, v₂ = velocities after collision

Conservation of kinetic energy

Because the collision is elastic, kinetic energy is also conserved.

(1/2)m₁u₁² + (1/2)m₂u₂² = (1/2)m₁v₁² + (1/2)m₂v₂²

Multiply by 2 to simplify:

m₁u₁² + m₂u₂² = m₁v₁² + m₂v₂²
Main idea: In an elastic collision, both momentum and kinetic energy are conserved.

2. Determine the velocities after collision

Step 1 – Start from momentum conservation

m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂

Step 2 – Use the elastic collision relation

For a one-dimensional elastic collision, the relative speed of separation equals the relative speed of approach:

u₁ - u₂ = -(v₁ - v₂)

So we can write:

u₁ - u₂ = v₂ - v₁

Step 3 – Express one final velocity through the other

v₂ = v₁ + u₁ - u₂

Step 4 – Substitute into the momentum equation

m₁u₁ + m₂u₂ = m₁v₁ + m₂(v₁ + u₁ - u₂)
m₁u₁ + m₂u₂ = (m₁ + m₂)v₁ + m₂u₁ - m₂u₂
(m₁ - m₂)u₁ + 2m₂u₂ = (m₁ + m₂)v₁
v₁ = ((m₁ - m₂)/(m₁ + m₂))u₁ + (2m₂/(m₁ + m₂))u₂

Step 5 – Find v₂

Use

v₂ = v₁ + u₁ - u₂

Then the result becomes

v₂ = (2m₁/(m₁ + m₂))u₁ + ((m₂ - m₁)/(m₁ + m₂))u₂
These are the standard formulas for a one-dimensional elastic collision.

3. Final formulas

Velocity of body 1 after collision

v₁ = ((m₁ - m₂)/(m₁ + m₂))u₁ + (2m₂/(m₁ + m₂))u₂

Velocity of body 2 after collision

v₂ = (2m₁/(m₁ + m₂))u₁ + ((m₂ - m₁)/(m₁ + m₂))u₂
Answer: These two formulas give the final velocities after any elastic head-on collision in one dimension.

4. Special case: m₁ = m₂

Let

m₁ = m₂ = m

Substitute into the formula for v₁:

v₁ = ((m - m)/(m + m))u₁ + (2m/(m + m))u₂
v₁ = 0·u₁ + 1·u₂ = u₂

Now for v₂:

v₂ = (2m/(m + m))u₁ + ((m - m)/(m + m))u₂
v₂ = 1·u₁ + 0·u₂ = u₁
Result for equal masses:
v₁ = u₂ v₂ = u₁
The two bodies exchange velocities.

Physical meaning

If two equal masses collide elastically, each one leaves with the other one’s initial velocity. This is the classic billiard-ball result.

If body 2 is initially at rest, then body 1 stops and body 2 moves away with body 1’s old speed.

5. Limit case: m₂ ≫ m₁

Now suppose the second body is much heavier than the first:

m₂ ≫ m₁

Then approximately:

m₁ + m₂ ≈ m₂ m₁ - m₂ ≈ -m₂

Approximation for v₁

v₁ = ((m₁ - m₂)/(m₁ + m₂))u₁ + (2m₂/(m₁ + m₂))u₂

So approximately:

v₁ ≈ -u₁ + 2u₂

Approximation for v₂

v₂ = (2m₁/(m₁ + m₂))u₁ + ((m₂ - m₁)/(m₁ + m₂))u₂

So approximately:

v₂ ≈ u₂
Result for a very heavy second body:
v₁ ≈ 2u₂ - u₁ v₂ ≈ u₂

If the heavy body is initially at rest

u₂ = 0

Then:

v₁ ≈ -u₁ v₂ ≈ 0

So the light body bounces backward with almost the same speed, while the heavy body almost does not move. This is like a ball bouncing from a wall.

6. Physical interpretation

Elastic collision

Both momentum and kinetic energy stay constant. The objects rebound without losing kinetic energy.

Equal masses

The objects exchange velocities. This is a very symmetric case.

Heavy target

The heavy body changes very little. The light body changes a lot and may bounce back.

Big picture: the mass ratio decides how strongly each body changes its motion after the collision.

8. Final answers

Conservation of momentum

m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂

Conservation of kinetic energy

(1/2)m₁u₁² + (1/2)m₂u₂² = (1/2)m₁v₁² + (1/2)m₂v₂²

Velocities after collision

v₁ = ((m₁ - m₂)/(m₁ + m₂))u₁ + (2m₂/(m₁ + m₂))u₂ v₂ = (2m₁/(m₁ + m₂))u₁ + ((m₂ - m₁)/(m₁ + m₂))u₂

Case m₁ = m₂

v₁ = u₂ v₂ = u₁

Case m₂ ≫ m₁

v₁ ≈ 2u₂ - u₁ v₂ ≈ u₂

If also u₂ = 0, then

v₁ ≈ -u₁ v₂ ≈ 0
This completes the solution and interpretation of the elastic one-dimensional head-on collision problem.
