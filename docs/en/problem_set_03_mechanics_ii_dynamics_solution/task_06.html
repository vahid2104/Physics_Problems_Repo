Problem 6 – Motion with Linear Drag

Explanation and solution of motion under a drag force F = -kv with simple steps and animation.

Problem Statement

The drag force is given by:

F = -kv

Initial conditions:

v(0) = v₀ x(0) = 0

We want to find:

  • The equation of motion and its solution
  • The limit of velocity as time goes to infinity
  • The comparison with motion without drag

1. Equation of Motion

From Newton’s second law:

m dv/dt = F

Since the drag force is:

F = -kv

Substitute into Newton’s law:

m dv/dt = -kv

Now divide by m:

dv/dt = -(k/m)v

This is the differential equation of motion.

2. Solve for the Velocity

We solve:

dv/dt = -(k/m)v

Separate the variables:

dv/v = -(k/m) dt

Integrate both sides:

∫(1/v) dv = ∫-(k/m) dt
ln|v| = -(k/m)t + C

Exponentiate:

v(t) = C₁ e^(-(k/m)t)

Use the initial condition v(0) = v₀:

v₀ = C₁

So the velocity is:

v(t) = v₀ e^(-(k/m)t)

3. Solve for the Position

We know:

dx/dt = v(t)

Substitute the velocity:

dx/dt = v₀ e^(-(k/m)t)

Integrate:

x(t) = ∫ v₀ e^(-(k/m)t) dt
x(t) = -(mv₀/k)e^(-(k/m)t) + C

Use the initial condition x(0) = 0:

0 = -(mv₀/k) + C
C = mv₀/k

Therefore:

x(t) = (mv₀/k) [1 - e^(-(k/m)t)]

4. Limit of the Velocity

We found:

v(t) = v₀ e^(-(k/m)t)

Now take the limit as t → ∞:

lim t→∞ v(t) = lim t→∞ v₀ e^(-(k/m)t)

Since the exponential term goes to zero:

lim t→∞ v(t) = 0

So the body slows down more and more, and eventually comes to rest.

5. Compare with Motion Without Drag

With Drag

F = -kv
v(t) = v₀ e^(-(k/m)t)
x(t) = (mv₀/k)[1 - e^(-(k/m)t)]
  • Velocity decreases exponentially
  • Velocity approaches zero
  • Position approaches a finite value

Without Drag

F = 0
v(t) = v₀
x(t) = v₀ t
  • Velocity stays constant
  • The body never slows down
  • Position increases linearly forever

Final Answers

The equation of motion is:

m dv/dt = -kv

Or equivalently:

dv/dt = -(k/m)v

The velocity is:

v(t) = v₀ e^(-(k/m)t)

The position is:

x(t) = (mv₀/k)[1 - e^(-(k/m)t)]

The limit of velocity is:

lim t→∞ v(t) = 0

Comparison:

  • With drag: the body slows down and stops
  • Without drag: the body keeps moving with constant velocity
