Explanation

The position point follows the parametric curve (x, y) = (t², sin t). The green arrow is the velocity vector, which points in the direction of motion. The orange arrow is the acceleration vector, which shows how velocity changes. The cross product has only a k-component here, so it points perpendicular to the x–y plane.
