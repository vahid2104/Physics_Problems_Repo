Part A — Given velocity

Given:

v(t) = (2t, 3, -e^(-t)) r(0) = (0, 1, 2)

Step 1: Position vector

r(t) = r(0) + ∫[0→t] v(τ) dτ
r(t) = (0,1,2) + ∫[0→t] (2τ, 3, -e^(-τ)) dτ
∫[0→t] 2τ dτ = [τ²][0→t] = t² ∫[0→t] 3 dτ = [3τ][0→t] = 3t ∫[0→t] -e^(-τ) dτ = [e^(-τ)][0→t] = e^(-t) - 1
r(t) = (0,1,2) + (t², 3t, e^(-t)-1) = (t², 1+3t, 1+e^(-t))

Step 2: Acceleration vector

a(t) = dv/dt = (d/dt(2t), d/dt(3), d/dt(-e^(-t))) = (2, 0, e^(-t))
Final A: r(t) = (t², 1 + 3t, 1 + e^(-t)) a(t) = (2, 0, e^(-t))

Part B — Given acceleration

Given:

a(t) = (4, -sin t, 0) v(0) = (1, 0, 2) r(0) = (0, 0, 0)

Step 1: Velocity vector

v(t) = v(0) + ∫[0→t] a(τ) dτ
v(t) = (1,0,2) + ∫[0→t] (4, -sin τ, 0) dτ
∫[0→t] 4 dτ = 4t ∫[0→t] -sin τ dτ = [cos τ][0→t] = cos t - 1 ∫[0→t] 0 dτ = 0
v(t) = (1,0,2) + (4t, cos t - 1, 0) = (1 + 4t, cos t - 1, 2)

Step 2: Position vector

r(t) = ∫[0→t] v(τ) dτ + r(0)
r(t) = ∫[0→t] (1+4τ, cos τ - 1, 2) dτ
∫[0→t] (1+4τ) dτ = [τ + 2τ²][0→t] = t + 2t² ∫[0→t] (cos τ - 1) dτ = [sin τ - τ][0→t] = sin t - t ∫[0→t] 2 dτ = 2t
r(t) = (t + 2t², sin t - t, 2t)
Final B: v(t) = (1 + 4t, cos t - 1, 2) r(t) = (t + 2t², sin t - t, 2t)

What this problem teaches

Velocity is the derivative of position. So position is found by integrating velocity.
Acceleration is the derivative of velocity. So velocity is found by integrating acceleration.
Initial conditions like r(0) and v(0) fix the constants that appear after integration.

Tip: in vector-valued motion problems, do everything component by component. That is usually the cleanest route.
