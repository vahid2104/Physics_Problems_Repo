Worked summary

These panels match the mathematical results for each of the four curves.

Curve A

Circle
$$x^2+y^2=R^2$$ $$\vec v(t)=(-R\sin t,\;R\cos t)$$ $$\vec a(t)=(-R\cos t,\;-R\sin t)$$ $$|\vec v|=R,\qquad |\vec a|=R$$

Both speed and acceleration magnitude are constant.

Curve B

Ellipse
$$\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$$ $$\vec v(t)=(-a\sin t,\;b\cos t)$$ $$\vec a(t)=(-a\cos t,\;-b\sin t)$$ $$|\vec v|=\sqrt{a^2\sin^2 t+b^2\cos^2 t}$$ $$|\vec a|=\sqrt{a^2\cos^2 t+b^2\sin^2 t}$$

In general, neither magnitude is constant unless $a=b$.

Curve C

Parabola
$$y=x^2$$ $$\vec v(t)=(1,\;2t)$$ $$\vec a(t)=(0,\;2)$$ $$|\vec v|=\sqrt{1+4t^2},\qquad |\vec a|=2$$

The speed changes with $t$, but the acceleration magnitude stays constant.

Curve D

Hyperbola
$$x^2-y^2=1$$ $$\vec v(t)=(\sinh t,\;\cosh t)$$ $$\vec a(t)=(\cosh t,\;\sinh t)$$ $$|\vec v|=\sqrt{\sinh^2 t+\cosh^2 t}$$ $$|\vec a|=\sqrt{\cosh^2 t+\sinh^2 t}$$

Neither magnitude is constant.
