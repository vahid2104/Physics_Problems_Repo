Problem 5 – Trajectory Curvature and Normal Acceleration

This page explains the ellipse motion in a simple way. Each animation can be shown separately, and at the end you can also show everything together.

x = a cos t,   y = b sin t
r(t) = (a cos t, b sin t)

1. Position on the ellipse

The point moves along the ellipse according to the parameter t. When t changes, the coordinates of the point also change.

x = a cos t
y = b sin t

So the position vector is:

r(t) = (a cos t, b sin t)

In the animation, the white point is the moving particle. The line from the center to the point is the position vector.

2. Velocity

Velocity tells us how the position changes with time. It gives both the speed and the direction of motion.

v(t) = dr/dt = (-a sin t, b cos t)
|v(t)| = √(a2 sin2 t + b2 cos2 t)

The velocity vector is always tangent to the ellipse. In the animation, it is shown by the green arrow.

3. Acceleration

Acceleration tells us how the velocity changes with time. It can change the speed, the direction, or both.

a(t) = dv/dt = (-a cos t, -b sin t)

In the animation, acceleration is shown by the yellow arrow.

4. Unit tangent vector

The unit tangent vector gives only the direction of motion. Its length is 1.

T̂(t) = v(t) / |v(t)|
T̂(t) = (-a sin t, b cos t) / √(a2 sin2 t + b2 cos2 t)

In the animation, it is shown by the blue arrow.

5. Tangential and normal acceleration

Acceleration can be divided into two parts:

a = at + an

Tangential acceleration changes the speed.

at = (v · a) / |v|
at = ((a2 - b2) sin t cos t) / √(a2 sin2 t + b2 cos2 t)

Normal acceleration changes the direction of motion.

an = |v × a| / |v|
an = ab / √(a2 sin2 t + b2 cos2 t)

6. Radius of curvature at t = 0

The radius of curvature shows how sharply the path bends.

an = v2 / R
R = v2 / an

At t = 0:

v(0) = (0, b)
|v(0)| = b
an(0) = a
R(0) = b2 / a
